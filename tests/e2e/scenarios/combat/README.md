# Combat Physical E2E Scenarios

This suite owns feature-specific deterministic combat scenarios consumed by the single Universal Physical E2E platform.

Combat manifests may name exact creatures and expected outcomes only when those values are backed by current repository fixture/spawn evidence. Nontrivial navigation must reuse the canonical `follow_route` contract. The suite does not own a second runner, pathfinder, OTBM parser, workflow, client fork, or generic creature-fixture interface.
